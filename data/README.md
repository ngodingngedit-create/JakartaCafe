Dataset cafe di Jakarta dalam format JSON lengkap dengan koordinat latitude & longitude.

## Dataset
Data tersedia di folder `data/`:
- Jakarta Timur
- Jakarta Barat
- Jakarta Utara
- Jakarta Pusat
- Jakarta All
## Field
Setiap cafe memiliki:
- nama_cafe
- rating
- jumlah_review
- kategori
- alamat
- latitude
- longitude
- google_maps_url

## Use Case
- Visualisasi peta
- Analisis lokasi
